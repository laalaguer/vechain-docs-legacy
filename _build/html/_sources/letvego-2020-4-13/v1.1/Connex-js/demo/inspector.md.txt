## What is Inspector?

Inspector is a static web site for contract call with ABI and contract address.

[Try it out!](https://inspector.vecha.in/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
