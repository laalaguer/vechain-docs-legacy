— title: “AWS Marketplace” slug: “aws” hidden: false createdAt: “2019-02-22T07:48:17.838Z” updatedAt: “2019-04-24T07:01:00.567Z” — It is easy to deploy VeChainThor node in AWS as there are AMIs for solo node, mainnet node and testnet node in the marketplace. And you can find it from the below link: https://aws.amazon.com/marketplace/search/results?x=0&y=0&searchTerms=vechain
