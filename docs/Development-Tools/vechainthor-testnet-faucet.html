What’s Faucet

TestNet Faucet provides VET and VTHO on VeChainThor testnet freely. It is designed to help developers test applications as well as provide users an alternative to experience dApps on the VeChainThor blockchain,

How to Claim VET and VTHO at Faucet

  1. Open Sync, and select testnet [block:image] { “images”: [ { “image”: [ “https://files.readme.io/57fca27-faucet-1.png”, “faucet-1.png”, 997, 696, “#f4f3f3” ] } ] } [/block]
  2. Type Faucet URL “http://faucet.vecha.in” into the address bar, and click “It’s me!!!” to claim VET and VTHO. Then sign the certificate of identification as required.
