VeKey - Secure Identity Device¶

[block:api-header] { “title”: “General Description” } [/block] The VeKey (device no.: VC-VK-U8001) is a device for digital signature and digital identity management. Data is stored in its persistent memory. The VeKey has no embedded battery, you can use a standard micro USB cable to connect to PC to power the VeKey. [block:image] { “images”: [ { “image”: [ “https://doc.vechainworld.io/images/fe26a3a-vekey.png”, “vekey.png”, 683, 383, “#d2c8c6” ] } ] } [/block]

[block:api-header] { “title”: “Product Features” } [/block]

[block:image] { “images”: [ { “image”: [ “https://doc.vechainworld.io/images/6a577c5-webwxgetmsgimg.jpg”, “webwxgetmsgimg.jpg”, 3750, 2500, “#e6e8e8” ] } ] } [/block]

  • Surface Material: Aluminium alloy, Sandblasting oxidation in surface
  • Device Dimensions: 58mm×32.1mm×6.6mm
  • Screen Dimensions: 0.96 inches OLED screen, 26.7mm×19.26mm×1.45mm
  • Color: Black, Red* (other colors requirements need contact VeChain)
  • Fingerprint Module: support
  • ESD Voltage Immunity: 6KV for contact, 10KV for air
  • Communication mode: USB
  • Language: English [block:api-header] { “title”: “Functions” } [/block]
  • Digital signature
  • Digital identity [block:api-header] { “title”: “Competitive Advantage” } [/block]
  1. Embedded with secure MCU endowed with CC EAL4+ certification, the UnionPay card chip product safety certification.
  2. Compulsory double authentication (Biology identification + PIN).
  3. Secure update procedure with firmware verification by VeChain.
  4. Secure transaction, which verifies certificates issued by VeChain when transact.
  5. Secure protection mechanism. [block:api-header] { “title”: “Ordering information” } [/block]
