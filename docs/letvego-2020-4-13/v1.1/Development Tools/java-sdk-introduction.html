— title: “Thor Java Client SDK” slug: “java-sdk-introduction” excerpt: “https://github.com/vechain/thor-client-sdk4j” hidden: false createdAt: “2019-02-22T03:49:27.680Z” updatedAt: “2019-04-09T12:12:57.820Z” — # What is Thor Java Client SDK?

A SDK toolkit for client to call VeChainThor Restful API.

Requires JDK8.

latest version 0.0.8

default package jar without dependencies,use -Pall to build all in one jar

## Get Started

### Create JAR

  • git clone https://github.com/vechain/thor-client-sdk4j
  • cd thor-client-sdk4j && mvn install

### Add JAR to project

Note: Here we demonstrate with IDEA, it is roughly the same in Eclipse.

  • File > Project Structure
  • Project Setting s > Modules > Dependencies > + > JARS or directories
  • Select the jar file and click on Open, then click OK button to confirm

## License Thor Java Client SDK is licensed under the GNU Lesser General Public License v3.0, also included in LICENSE file in repository. - - - -

## Set blockchain nodes provider Set blockchain nodes sample as follows

` NodeProvider nodeProvider = NodeProvider.getNodeProvider(); nodeProvider.setProvider("http://localhost:8669"); nodeProvider.setTimeout(10000); `

## You can find the clients toolkit under the directory :
src/main/java/com/vechain/thorclients/clients

There are some SDK specifications <a href=”https://github.com/vechain/thor-client-sdk4j/blob/dev/doc”> sdk4j doc </a>

For JUnit Test example: there are some required parameters in config.properties file to support JUnit Test running.
