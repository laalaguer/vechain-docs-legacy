— title: “Connex Framework” slug: “connex-framework” excerpt: “URL: https://github.com/vechain/connex-framework” hidden: false createdAt: “2019-08-16T07:33:06.395Z” updatedAt: “2019-08-16T07:36:05.852Z” — Connex Framework is a library implements Connex interface. It helps various wallet instances offer consistent Connex interface to VeChain DApps.

## Installation

`sh npm i @vechain/connex-framework `

## Usage

To create framework instance, Connex.Driver needs to be implemented

```typescript import { Framework } from ‘@vechain/connex-framework’ import ‘@vechain/connex.driver’

class MyDriver implements Connex.Driver {
// implementations

}

const driver = new MyDriver()

// it’s suggested in development mode, which is helpful to diagnose driver implementation. // const framework = new Framework(Framework.guardDriver(driver))

const framework = new Framework(driver)

// here framework is the ready-to-use Connex instance object ```
