### Golang environment

Thor requires Go 1.10+ and C compiler to build. To install Go, follow this link.(Thor requires Go 1.10+ and C compiler to build. To install Go, follow this [link](https://golang.org/doc/install))

#### Get Thor source code

  1. before this step, make sure your Golang can be work on your computer

  2. clone the Thor source code from Github to your GOPATH directory

    ` git clone https://github.com/vechain/thor.git cd thor `

#### Visual Studio Code

  1. Download and Install VSCode from [Link](https://code.visualstudio.com/)
  2. Open VSCode and click the Extensions button (on the left of the window)
  3. Search ‘Go’ extension, it’s publisher is Microsoft
  4. Install the extension and restart VSCode

### Import Thor source code to VS Code

  1. Open VSCode and click File -> open
  2. Open the Thor source code root directory (eg. GOPATH/src/thor)
